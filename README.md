# crop_diagonally
1. File Crop_image.py contain two functions. Function **crop_diagonal()** help you crop diagonally (rotated retangle) the image. Function **normal_crop()** help you crop the image with axis-aligned retangle. <br>
2.  Please read the comment in the code to understand how to use it, very easy. <br>
Code này sẽ giúp crop chéo ảnh (crop theo hình chữ nhật được xoay) và cắt thường (hình chữ nhật có các cạnh song song với trục x,y của ảnh). Hãy đọc các comment trong code để hiểu cách sử dụng, rất dễ.
